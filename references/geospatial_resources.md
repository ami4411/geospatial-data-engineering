# Geospatial References

## Datasets worth knowing

- **Natural Earth** for small-scale global boundaries and basemaps
- **OpenStreetMap** for volunteered geographic information and network features
- **Copernicus Sentinel** for Earth observation imagery and environmental monitoring
- **Overture Maps** as an increasingly relevant open map data initiative

## Reading list

- *Spatial Data Science* by Luc Anselin
- *Geographic Data Science with Python* by Bonham-Carter, Brunsdon, Comber and others
- PostGIS documentation for practical spatial SQL
- GeoParquet specification for modern vector storage

## Standards and ideas to keep an eye on

- STAC for catalogue design and discovery
- Cloud Optimized GeoTIFF for raster access patterns
- GeoParquet for analytical vector workflows
